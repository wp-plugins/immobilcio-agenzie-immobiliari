== Description ==

[ENG]
**ImmobilCIO gives new features on WordPress for Real Estate Agency.**

This plugin creates one custom post type for properties, one metabox, two taxonomies and gives you a widget where you display last property published and a options page.

Immobilcio insert on single post:
*	all details saved on metabox.
*	Google Maps (on options page you can configure width and heigth for map)

ImmobilCIO is multilanguages : you can create your localization and put it in `/immobilicio/lang/` directory with this example name: immobilcio-it_IT

On options page you can activate or deactivate display maps on single post.

New features are coming for next release.

[ITA]
**ImmobilCIO fornisce nuove caratteristiche sul pannello WordPress per le Agenzie Immobiliari.**

Questo plugin utile per le **Agenzie Immobiliari** che volessero utilizzare WordPress per il proprio sito web, crea un nuovo custom post type, un metabox per l' inserimento dettagli sul singolo immobile e due box tassonomie per categorizzare l' immobile in fasce di prezzo ed in zone.

ImmobilCIO supporta la localizzazione e quindi può essere tradotto in più lingue. In `/immobilicio/lang/` verranno inserite le varie traduzioni.

Nella pagina Opzioni di **ImmobilCIO** puoi attivare/disattivare la visualizzazione della Google Maps nei singolo post.

Nuove funzionalità arriveranno nelle prossime release.

== Installation ==

1. Upload `immobilcio` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy with ImmobilCIO

== Frequently Asked Questions ==

= I get 404 error when I use dropdown menu on widget. Solution? =

You can try to disable and re-enable the permalink.

= Is possible with some templates that the widget overflows from the sidebar? =

Yes, but you can solve this problem: you can add new style of `immobsingle` ID on your style.css.
In the next release you will be able to manage the style from the options page.

= How I manage gallery on single custom post type? =

Now you can use Add Media button on editor page.
CMS-Italia.org is thinking about a new feature to make it easier to manage your photos.

= Is ImmobilCIO compatible with WPMU? =

Alredy ImmobilCIO isn't tested on WIPMU.

= All fields are required METABOX? =

No, but if you enter them all will be beneficial to the search engines and all the functionality of the plugin will work fine
